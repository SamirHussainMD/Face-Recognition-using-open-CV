# Face Recogniton GUI-APP



# A very Simple Gui app for Face Detection 

  - Collect Face Data
  - Build Face Classifier 
  - Detecte the face

#  New Features!

  - Add Emotion detection
  - Fixed image loading in Linux 
  
  
# Installation

1 : Install the requirements .

```sh
$ pip install -r  requirements.txt
```

2 : Run The App 

```sh
$ python app-gui.py
```
